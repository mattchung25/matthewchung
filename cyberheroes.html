Cyberheroes

JavaScript CSS Full Stack Development

Project Description

Interactive website that teaches 3rd and 4th graders about cybersecurity using games and quizzes. The project focuses on making cybersecurity education engaging and making it accessible for young learners. The website does not store user data and is supposed to be an open world for users to explore the lessons and topics they want to learn about.

Key Features:

  • Educational games and interactive quizzes about cybersecurity
  • Age-appropriate content designed for 3rd and 4th graders
  • Animations throughout the website to keep the user engaged
  • Accessibility features including things like line readers and underlined words.

The project combines educational content with interactive elements to create an effective learning experience for young students about important cybersecurity concepts.
