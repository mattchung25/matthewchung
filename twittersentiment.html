Twitter Sentiment Analysis

Python Machine Learning NLP

Project Description

In this NLP project, my team and I explored sentiment analysis and named entity recognition (NER) on Twitter data to understand emotional tone and entity popularity. Using datasets like Sentiment140, Twitter Airline Sentiment, and Large Twitter Tweets Sentiment, we applied multiple modeling approaches, including:

Key Features:

  • Logistic Regression with TF-IDF vectors (Best performer: F1 score ~0.78)
  • k-Nearest Neighbors (kNN) using cosine similarity
  • Fine-tuned DistilBERT for efficient transformer-based classification
  • spaCy’s NER model to extract and analyze named entities

We found that tweet sentiment generally became more positive with increased tweet length, and certain entities like “#followfriday” were highly associated with positive sentiment. Logistic regression outperformed the other models and was used to evaluate sentiment trends across entities.

Growth

What I Did:

I implemented a k-Nearest Neighbors (kNN) model with cosine similarity to analyze sentiment and NER on Twitter data. Due to the high-dimensional, sparse nature of the dataset, cosine similarity was chosen over Euclidean distance, and an n-value of 5 (instead of the standard 3) was used to reduce variance. However, kNN performed the worst compared to other models because of its lazy learning approach and reliance on surface-level feature similarity, which limits its ability to capture contextual meaning in text.

What I Learned:

  • Gained experience in natural language processing (NLP) and text mining techniques
  • Explored and compared the performance of multiple machine learning models for sentiment analysis
  • Developed hands-on skills in data cleaning, data transformation, and training models on large datasets
